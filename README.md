## Arrow Judge

**Arrow Judge** is open source online judge system.

## Requirements
 * Python 2.7.x

## Installation
 * Install module with `python setup.py install`
 * Copy server.conf to /etc/arrow-judge
 * Run with `arrow-judge start`
